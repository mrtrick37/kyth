# Works Better Here

KythOS should not pitch Linux as a compromise. The goal is a gaming OS that
feels calmer, faster to recover, and easier to trust than a typical Windows
gaming install.

## User-Facing Wins

- Atomic updates: OS upgrades stage cleanly and roll back as a whole image.
- No surprise Windows Update restart in the middle of a game night.
- GameMode, sched-ext, MangoHud, Gamescope, vkBasalt, and GE-Proton are ready
  before the first game install.
- Per-game launch profiles make HDR, VRR, FPS caps, and troubleshooting options
  copyable instead of mysterious.
- Steam libraries can be copied from Windows drives read-only.
- Ludusavi gives players a save backup path before risky migration or modding.
- The Compatibility page marks publisher anti-cheat blocks explicitly instead
  of blaming Linux vaguely.

## Proof Points to Keep Fresh

- Number of smoke-test games validated for the current image.
- Current GE-Proton version.
- Current kernel and Mesa/NVIDIA driver versions.
- Known blocked games with source check dates.
- Real save migration and modding validation results.
